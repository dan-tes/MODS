# MODS
Simulation of the motion of a material point in two-dimensional space. Using this program, you can visualize the mechanical movement of a body in space by experimenting with different initial data. This product can be used by: teachers and students, teachers and students, to explain and improve knowledge on the topic of the project. The Java language was chosen because I am proficient in most of its constructions. The framework for working with graphics is Swing.
